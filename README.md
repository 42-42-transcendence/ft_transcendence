## Dockerfile & Makefile을 이용하여 빌드

- 프론트 리액트 테스트 서버를 빌드합니다.
- 실제로는 저희는 백엔드에서 프론트를 리액트로 빌드하여 index.html을 배포해야 합니다.

---

### Get Started

- start
  - make start
- open
  - http://localhost:3001/login
- clear
  - make fclean

---

### make

- 이미지를 빌드합니다.

### make start

- 컨테이너를 실행시킵니다.

### make clean

- 컨테이너를 지웁니다.

### make fclean

- 이미지와 컨테이너를 지웁니다.

### make re

- 컨테이너를 지우고 다시 실행시킵니다.

### make fre

- 이미지와 컨테이너를 지우고 다시 실행시킵니다.
